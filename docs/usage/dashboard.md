# ダッシュボード

**URL**: `/dashboard`

現在進行中のスプリントの概要を一目で確認できるトップ画面です。

---

## 表示内容

### メトリクスカード（4枚）

| カード | 内容 |
|---|---|
| 総ポイント | スプリント内の全 Issue のストーリーポイント合計 |
| 完了ポイント | クローズされた Issue のストーリーポイント合計 |
| 残ポイント | 総ポイント − 完了ポイント |
| 残日数 | 今日からスプリント終了日までの日数 |

### バーンダウンチャート

スプリント期間中のポイント消化状況をグラフで表示します。

- **理想線（グレー点線）**: 毎日均等にポイントが消化された場合の推移
- **実績線（青実線）**: 実際の Issue クローズ日時に基づく残ポイントの推移

> 実績線はまだ到来していない日付には表示されません。

### ベロシティトレンド

直近スプリントのベロシティ（完了ストーリーポイント）を棒グラフで表示します。
スプリント計画時の参考ベロシティとして活用できます。

### KPT サマリー

現在のスプリントに登録されているレトロスペクティブ（KPT）の件数を表示します。

| 種別 | 説明 |
|---|---|
| Keep | うまくいっていること |
| Problem | 問題・改善が必要なこと |
| Try | 次に試してみること |

### 進行中 Issue

現在のスプリントでまだオープンな Issue の一覧です。Issue 番号・タイトル・担当者・ストーリーポイントを表示します。

### ブロッカー一覧

現在のスプリントでブロッカー（`is_blocker = true`）が設定されている Issue を赤いカードで表示します。

| 表示項目 | 説明 |
|---|---|
| Issue 番号・タイトル | GitHub の Issue 番号とタイトル |
| リポジトリ名 | 所属リポジトリ |
| 担当者 | アサインされたユーザー名 |
| 理由 | ブロッカーとして登録された理由 |

ブロッカーがない場合はこのセクションは表示されません。

### PR レビュー待ち一覧

現在のスプリントでまだマージ・クローズされていない PR のうち、レビュー待ち（Open / Changes 状態）の一覧を黄色いカードで表示します。

| 表示項目 | 説明 |
|---|---|
| PR 番号・タイトル | GitHub の PR 番号とタイトル |
| リポジトリ名 | 所属リポジトリ |
| 担当者 | PR の担当者 |
| GitHub リンク | PR ページへの外部リンク（🔗アイコン） |

レビュー待ち PR がない場合はこのセクションは表示されません。

---

## 「現在のスプリント」の決定ロジック

- `state = 'open'` のスプリントのうち、**終了日が最も近いもの**を「現在のスプリント」として表示します
- 進行中のスプリントがない場合は、全てのカードに 0 または空が表示されます

---

## よくある質問

**Q: バーンダウンチャートが表示されない**
→ スプリントに「開始日」が設定されているか確認してください。開始日はスプリント設定から手動で入力します。

**Q: メトリクスがすべて 0 になる**
→ Issue にストーリーポイントが設定されていない可能性があります。スプリント詳細画面から各 Issue にポイントを設定してください（今後の機能追加予定）。
