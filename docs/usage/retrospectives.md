# レトロスペクティブ

**URL**: `/retrospectives`

スプリントごとの振り返り（KPT）を記録・管理する画面です。

---

## スプリント選択

画面右上のドロップダウンでスプリントを選択します。進行中のスプリントには `(進行中)` のラベルが付きます。

スプリントを切り替えると、そのスプリントの KPT 一覧が表示されます（URL の `?sprint_id=` パラメータで管理）。

---

## KPT の追加

スプリントを選択すると入力フォームが表示されます。

| 入力項目 | 説明 |
|---|---|
| 種別（ラジオ） | `Keep` / `Problem` / `Try` から選択 |
| 内容 | テキストエリアに内容を入力（必須） |

「追加」ボタンで登録します。

---

## KPT ボード

登録済みの KPT は種別ごとに3列のカードボードで表示されます。

| 種別 | 色 | 説明 |
|---|---|---|
| Keep | 緑 | うまくいっていること・継続したいこと |
| Problem | 赤 | 問題・課題・改善が必要なこと |
| Try | 青 | 次のスプリントで試してみること |

---

## KPT の編集・削除

各カードの「編集」ボタンからインライン編集ができます。

- **編集**: テキストエリアが展開され、「保存」で確定、「キャンセル」で元に戻ります
- **削除**: 確認ダイアログが表示され、確認後に削除されます

---

## よくある質問

**Q: スプリントが選択できない**
→ GitHub 同期でマイルストーンを取り込んでください。スプリントは GitHub マイルストーンから自動作成されます。

**Q: 過去のスプリントの KPT も見たい**
→ ドロップダウンから過去のスプリントを選択してください。全スプリント分が選択可能です。
