# マイルストーン

**URL**: `/milestones`

GitHub マイルストーンの一覧を表示します。マイルストーンは中長期ゴールの管理に使用し、スプリントとは独立して表示されます。

---

## 表示内容

各マイルストーンには以下の情報が表示されます。

| 項目 | 説明 |
|------|------|
| タイトル | マイルストーン名 |
| 期限 | `due_on`（未設定の場合は「期限なし」） |
| ステータス | `open`（緑バッジ）/ `closed`（グレーバッジ） |
| リポジトリ | 所属リポジトリ名（GitHub へのリンク付き） |

マイルストーンはリポジトリごとにグループ表示されます。

---

## Iteration モードとの関係

リポジトリに `GitHub Project Number` を設定している場合（Iteration モード）、スプリントは GitHub Projects の Iteration から同期されます。マイルストーンはスプリントとは連携せず、このページで独立して確認できます。

`GitHub Project Number` が未設定の場合（Milestone モード）、マイルストーンはスプリントとして同期されるため、このページの内容はスプリント一覧とほぼ同じになります。

---

## データの更新

マイルストーンはナビバーの「GitHub 同期」ボタンを押すことで最新状態に更新されます。
