# マイルストーン

**URL**: `/milestones`

マイルストーンはプロジェクトの中長期ゴール・目標を管理します。年月単位で整理され、配下のスプリント一覧・集計が表示されます。

> **自動生成**: `/milestones` にアクセスするたびに、現在月 −6ヶ月 〜 現在月 +12ヶ月（合計19ヶ月）のマイルストーンが自動補完されます。手動での作成・削除はできません。

---

## ページ構成

### 一覧ページ（`/milestones`）

「今月・今後」タブ（デフォルト）と「過去」タブの2タブ構成で表示します。

| タブ | 表示対象 | 並び順 |
|------|---------|--------|
| 今月・今後 | 今月以降のマイルストーン | 年月昇順（近い月が上） |
| 過去 | 先月以前のマイルストーン | 年月降順（直近の過去が上） |

各マイルストーンには以下の情報が表示されます。

| 項目 | 説明 |
|------|------|
| 年月 | `2026年4月` などの形式 |
| タイトル | マイルストーン名 |
| ステータス | `planning`（青）/ `in_progress`（黄）/ `done`（緑） |
| 期限 | `due_date`（未設定の場合は「—」） |
| スプリント数 | 紐付けられているスプリント件数 |

**アクション**
- **詳細・編集**: 各マイルストーン行をクリック → `/milestones/{id}` に遷移

### 詳細ページ（`/milestones/{id}`）

マイルストーンの詳細情報と配下のスプリント一覧・集計を表示します。

**表示内容**

| 項目 | 説明 |
|------|------|
| 年月 | `2026年4月` |
| タイトル | マイルストーン名 |
| ゴール | 説明文（あれば） |
| ステータス | 現在のステータス（`planning` / `in_progress` / `done`） |
| 着手日 | 実際の開始日（未設定なら「—」） |
| 期限 | マイルストーン期限日 |

**配下スプリント一覧**
- 開始日順に表示
- 各スプリントの課題数・ベロシティ・バーンダウン情報を表示
- スプリント行をクリックで詳細表示

**アクション**
- **編集**: 「編集」ボタン → `/milestones/{id}/edit` に遷移
- **スプリント紐付け**: 「スプリントを追加」ボタン → 未割当スプリント一覧モーダルが開く
  - モーダルでスプリント選択 → `PATCH /sprints/{sprint}/milestone` で紐付け
- **スプリント解除**: 各スプリント行の「解除」ボタン → `PATCH /sprints/{sprint}/milestone` で `milestone_id = NULL` に設定

### 編集ページ（`/milestones/{id}/edit`）

既存マイルストーンのタイトル・ゴール・ステータス・日付を編集するフォームです。

| フィールド | 型 | 説明 |
|-----------|---|------|
| タイトル | text | マイルストーン名（必須） |
| ゴール | textarea | 目標・説明（オプション） |
| ステータス | select | `planning` / `in_progress` / `done` |
| 着手日 | date | 実際の開始日（オプション） |
| 期限 | date | リリース予定日（オプション） |

> 年・月は自動生成で決定されるため、編集フォームでは変更できません。削除機能もありません。

---

## スプリントとの関係

マイルストーンは複数のスプリントを紐付けられます。

- **紐付け方法**: マイルストーン詳細ページ → 「スプリントを追加」ボタン → 未割当スプリント一覧から選択
- **解除方法**: マイルストーン詳細ページの各スプリント行 → 「解除」ボタン
- **手動管理**: GitHub 同期では自動的に紐付きません

スプリント削除時は `milestone_id` が自動的に NULL に設定されます。

## マイルストーン管理の特徴

### 自動生成

`/milestones` にアクセスするたびに `MilestoneGeneratorService` が実行され、以下の範囲のマイルストーンを `firstOrCreate` で冪等に補完します。

| 範囲 | 内容 |
|------|------|
| 過去側 | 現在月 −6ヶ月 |
| 未来側 | 現在月 +12ヶ月 |
| 合計 | 19ヶ月分 |

- 毎月自動的にスライドするため、「常に今月から12ヶ月先が存在する」状態を維持します。
- 既存レコードは変更されません（冪等）。
- デフォルト値: タイトル=`{year}年{month}月`、ステータス=`planning`

### GitHub との独立性

- **GitHub との同期なし**: マイルストーンは GitHub と完全に独立して管理
- 旧仕様の「GitHub Monthly Iteration → Milestone 同期」は削除されました
- GitHub Projects の Iteration モード・Milestone モードとは関係なく独立して管理されます

### アプリ独自管理

- 年月の一意制約（`UNIQUE(year, month)`）により、同じ年月のマイルストーンは1件のみ
- ステータス・ゴール・期限は編集ページから自由に変更可能
- 手動作成・削除はできません（自動生成のみ）
