# PR 自動作成

以下の手順でPRを作成してください。

## 手順

### 1. ブランチ・差分を確認

```bash
git log main..HEAD --oneline
git diff main...HEAD --stat
```

### 2. PRタイトル・本文を作成

差分コミット全体を読んで以下のフォーマットで日本語PRを作成する。

**タイトル**: `<type>: <変更の概要>` （70文字以内）

**本文テンプレート**:
```markdown
## 概要

<1〜3行で変更の目的・背景>

## 主な変更内容

<変更を箇条書き。機能追加・バグ修正・リファクタ・環境整備などカテゴリ別にまとめる>

## 動作確認方法

<テスト実行コマンドや手動確認手順>
```

### 3. PR作成

```bash
gh pr create --title "<タイトル>" --body "<本文>"
```

### 4. PR URL を報告

作成されたPR URLをユーザーに伝える。
